## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/hudaputrasantosa/todolist-with-mevn
~~~

Go to the project directory  

~~~bash  
  cd todolist-with-mevn
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run start
~~~
